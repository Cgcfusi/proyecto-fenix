# Decisiones del Proyecto Fénix

---

## 16/07/2026

### ¿Por qué usamos n8n Cloud?

Porque queremos validar el negocio antes de administrar un servidor.

Cuando el proyecto genere ingresos migraremos a un VPS propio.

---

### ¿Por qué usamos Gemini Flash?

Porque el plan gratuito permite construir el MVP.

Cuando el volumen aumente evaluaremos modelos más potentes.

---

### ¿Por qué usamos GitHub desde el primer día?

Porque queremos mantener el proyecto ordenado y versionado desde el inicio.
